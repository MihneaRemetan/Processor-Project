This repository contains the complete structural Verilog implementation of a modular processor architecture developed for academic purposes. The design integrates a clearly separated datapath and control structure, including an Arithmetic Logic Unit (ALU), Control Unit, register modules, memory components, input/output subsystems, and a System-on-Chip (SoC) integration layer. The architecture follows a hierarchical modular methodology, enabling independent verification of each hardware block prior to full system integration. Functional validation is performed through dedicated testbenches, with simulation-driven waveform analysis supporting signal-level inspection of datapath activity and control flow behavior. The objective of the project is to demonstrate a scalable, maintainable, and systematically verified processor design using structural hardware description techniques. Additional architectural details, design assumptions, and verification procedures are provided in the accompanying documentation.
